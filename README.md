# Opinione360

Progetto-M24 del gruppo MRPL

## Descrizione generale 
Opinione360 Ã¨ un sistema di consultazione basato su un'architettura client-server che permette ad aziende o enti pubblici di creare votazioni o sondaggi rivolti a gruppi di persone selezionate di volta in volta (che devono essere preventivamente registrate con una propria e-mail, una password e un nome utente). Per un maggior livello di sicurezza le votazioni richiedono al votante di autenticarsi tramite un servizio esterno che ne permette l'identificazione.
Gli utenti del sistema sono divisi in tre categorie: amministratore (si occupa della creazione delle consultazioni), candidato o votante

## Tecnologie utilizzate
- Java
- MYSQL 8.0

## Dipendenze
Il sistema utilizza il Connector/J per il collegamento al DB
